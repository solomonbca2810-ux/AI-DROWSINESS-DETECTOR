# 🧠 AI Drowsiness Detection System

This project detects driver drowsiness in real-time using computer vision techniques and alerts the user with voice and alarm.

## 🚀 Features
- Real-time face and eye tracking
- Eye Aspect Ratio (EAR) based detection
- Continuous beep alert system
- Voice warning using pyttsx3
- Smooth performance using threading

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy

## ⚙️ Installation

pip install opencv-python mediapipe numpy pyttsx3 pypiwin32
